# 🎨 Shashank Sharma - Personal Portfolio

> A bold, modern, and interactive personal portfolio website built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies - just raw web development skills.

## ✨ Features

- **🎯 Pure Vanilla Stack**: Built entirely with HTML5, CSS3, and JavaScript - zero frameworks
- **🎨 Modern Design**: Eye-catching collage-style layout with bold typography
- **📱 Responsive Layout**: Fully responsive design that works on all devices
- **⚡ Lightning Fast**: No heavy frameworks means ultra-fast load times
- **🖼️ Visual Storytelling**: Creative image collage showcasing design work
- **📄 Resume Integration**: Direct PDF resume download functionality
- **✨ Smooth Interactions**: Custom JavaScript animations and interactions
- **🎭 Bold Typography**: Statement-making headline design

## 🚀 Quick Start

### Installation

```bash

```

## 📝 Usage

### Running Locally

1. **Double-click** `index.html`, or
2. **Open with Live Server** in VS Code for development
3. **Deploy anywhere** - No special server requirements needed

### Customization

#### Update Content
Edit `index.html` to change:
- Name and headline
- Profile information
- Resume link

#### Modify Styling
Edit `style.css` to customize:
- Colors and themes
- Layout and spacing
- Animations and effects

#### Add Functionality
Edit `script.js` to add:
- Interactive features
- Animation triggers
- Navigation logic

### Adding Your Own Images

Replace images in the `images/` folder:
```bash
images/
├── your-photo.jpg
├── project1.jpg
├── project2.jpg
└── design-work.jpg
```

Update image references in `index.html`.
